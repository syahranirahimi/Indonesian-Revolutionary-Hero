# Indonesian Revolutionary Hero Learning Application

An interactive Android-based learning application dedicated to introducing and honoring Indonesian revolutionary heroes. Developed as an academic project using **Java** and **Android Studio**, this application aims to deliver an engaging educational experience through a clean and user-friendly interface.

## 🚀 Features
* **Hero Profiles & History**: Comprehensive information and historical context about Indonesian revolutionary heroes.
* **Interactive UI**: Designed with an intuitive layout to make learning history more engaging for students and tech enthusiasts alike.
* **Optimized for Android**: Lightweight application compiled directly into an executable APK.

## 🛠️ Tech Stack
* **IDE**: Android Studio
* **Programming Language**: Java
* **Platform**: Android

## 📥 How to Download and Install

Since this application is currently in the development phase, you can download and install the prototype directly onto your Android device by following these steps:

### Step 1: Download the APK
1. On this GitHub repository page, look at the file list above.
2. Click on the file named **`app-debug.apk`**.
3. Click the **Download** button (usually represented by a download icon or a raw file download option) to save the file to your device.

### Step 2: Enable Unknown Sources (If Required)
Because this app is installed outside the Google Play Store, your device might ask for permission:
1. Open your device **Settings**.
2. Go to **Security** or **Privacy**.
3. Enable the option to **"Allow installation of apps from unknown sources"** (or grant permission to your browser/file manager to install APKs).

### Step 3: Install and Run
1. Open your device's **File Manager** and navigate to the **Downloads** folder.
2. Tap on **`app-debug.apk`**.
3. Click **Install** and wait for the process to complete.
4. Launch the **Indonesian Revolutionary Hero** app from your app drawer and enjoy learning!
